# StockSystem

<h2>Overview</h2> 
This stock trading application provides users to trade(buy/sell) the stock online and keep tracking of their transactions. 

<h2>Techs</h2>
FrontEnd : React, node.js
<br>
BackEnd : Spring boot, Spring mvc, Spring JPA
<br>
DB : Embedded H2
